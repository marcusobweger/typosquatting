# Phishing / Typosquatting Checker

Check if an URL is likely a phishing link

## Demo

Try the tool at https://phishingchecker.vercel.app

## About

Built with [Next.js](https://nextjs.org), designed with [shadcn](https://ui.shadcn.com) and hosted
on [Vercel](https://vercel.com)
